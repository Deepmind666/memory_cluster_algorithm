## 本案建议保留的核心耦合点
1. 异构碎片模型（含 provenance/backrefs）
2. 增量聚类 + 簇内融合
3. 冲突显式记录（非覆盖）
4. 偏好向量贯穿写入/压缩/降级/检索
5. 可逆展开路径（摘要 -> backrefs -> 原始证据）
