# 06 权利要求书草案（收敛版）

## 一、独立权利要求

### 权利要求1（方法）
一种多 Agent 协同记忆压缩与冲突治理方法，其特征在于，包括：
1. 获取多个代理写入的记忆碎片，所述碎片至少包括内容、来源代理标识、时间戳和结构化槽位信息；
2. 对所述记忆碎片进行语义向量化并执行增量聚类，得到多个语义簇；
3. 在簇内检测同槽位多值冲突，构建冲突证据图并计算冲突优先级；
4. 基于冲突密度、来源分布特征和时效衰减分配簇级保留预算；
5. 在簇合并阶段采用语义相似度通道与冲突兼容通道的双通道门控，仅在两个通道同时满足阈值时允许合并；
6. 在冲突语义抽取阶段保留条件作用域、反事实作用域和否定语义，并执行跨句代词回指解析；
7. 生成簇摘要和可逆回溯索引，以支持簇级检索与按需证据展开。

### 权利要求2（系统）
一种多 Agent 协同记忆压缩与冲突治理系统，其特征在于，包括：
- 碎片采集模块；
- 语义向量化模块；
- 增量聚类模块；
- 冲突证据图构建模块；
- 自适应预算模块；
- 双通道合并门控模块；
- 语义抽取与回指解析模块；
- 索引与检索模块；
其中各模块协同执行权利要求1所述方法。

### 权利要求3（存储介质）
一种计算机可读存储介质，其上存储有计算机程序，所述程序被处理器执行时实现权利要求1所述方法。

## 二、从属权利要求

### 权利要求4
根据权利要求1所述方法，其中所述冲突证据图至少包括：冲突值节点、值切换边、证据计数字段和主导值字段。

### 权利要求5
根据权利要求4所述方法，其中冲突优先级由冲突值数量、值切换次数和证据数量的加权组合计算得到。

### 权利要求6
根据权利要求1所述方法，其中所述自适应预算至少基于冲突密度、来源分布熵和时效衰减因子联合计算。

### 权利要求7
根据权利要求6所述方法，其中预算系数经限幅处理后用于调整簇摘要细节配额。

### 权利要求8
根据权利要求1所述方法，其中语义相似度通道与冲突兼容通道分别具有独立阈值，并采用“与”逻辑共同控制合并决策。

### 权利要求9
根据权利要求8所述方法，其中冲突兼容度基于槽位值集合交并关系计算，当兼容度为零时阻断合并。

### 权利要求10
根据权利要求1所述方法，其中语义抽取对条件句和反事实句执行作用域隔离，对否定表达执行否定窗口识别。

### 权利要求11
根据权利要求10所述方法，其中跨句代词回指至少支持中英文代词映射到最近有效槽位。

### 权利要求12
根据权利要求1所述方法，其中在簇合并阶段执行相似度上界剪枝，当理论上界低于合并阈值时跳过精确相似度计算。

### 权利要求13
根据权利要求1所述方法，其中可选地建立签名桶候选邻接关系，以减少需精确比较的候选簇对数量。

### 权利要求14
根据权利要求13所述方法，其中可选地采用多表近似签名候选检索，并与所述签名桶候选邻接关系进行联合门控。

## 三、撰写说明
- 核心主张建议集中在权利要求1、4-11。
- 权利要求12-14建议作为可选实施例，不作为当前版本答复审查意见的主防线。

## 四、当前答复优先级（与最新实验对齐）
1. 第一优先级（核心独立主线）
- 权利要求1（方法总流程）
- 权利要求4-11（CEG/ARB/DMG/语义精度）

2. 第二优先级（可选实施例）
- 权利要求12（Prune）：在稀疏场景收益明确，active 场景收益不稳定。
- 权利要求13-14（Candidate/ANN）：当前 active 场景仍存在负加速，且 Candidate 有有损风险，仅建议作为备选实施例保留。
