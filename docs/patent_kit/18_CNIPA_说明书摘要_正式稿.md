# 18 CNIPA 说明书摘要正式稿

## 说明书摘要
本发明公开了一种多智能体协同下的语义记忆碎片聚类压缩与冲突治理方法、系统及存储介质，属于人工智能与数据处理技术领域。该方法对多源记忆碎片执行语义向量化与增量聚类，在簇内构建冲突证据图并计算冲突优先级；基于冲突密度、来源分布和时效衰减分配簇级自适应保留预算；在簇合并阶段采用语义相似度通道与冲突兼容通道的双通道门控，降低语义近似但事实冲突的误合并；在语义抽取阶段执行条件作用域隔离、反事实作用域隔离、否定识别和跨句回指解析；并通过可逆回溯索引实现摘要到原始证据的按需展开。与现有滚动摘要方案相比，本发明可提升跨智能体记忆一致性、冲突可解释性和证据可审计性，并降低上下文注入开销。
