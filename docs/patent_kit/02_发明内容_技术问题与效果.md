# 02 发明内容：技术问题与技术效果

## 一、拟解决的技术问题
本发明旨在解决多 Agent 共享记忆治理中的以下技术问题：
- 碎片化信息大量重复，导致存储体量和检索成本上升。
- 冲突事实被隐式覆盖，导致后续代理读取错误版本。
- 偏好策略仅作用于摘要格式，无法控制写入与降级路径。
- 压缩后缺乏可逆索引，难以审计与回溯。

## 二、技术方案要点
- 以异构记忆碎片为最小单元建立统一数据模型。
- 对新写入碎片执行增量语义聚类。
- 在簇内先结构化融合，再执行摘要压缩。
- 对互斥槽位建立冲突记录，不做静默覆盖。
- 引入偏好向量控制保留强度与细节预算。
- 为摘要保留 backrefs，实现摘要到证据的可逆展开。

## 三、技术效果
- 降低共享记忆条目冗余与存储占用。
- 降低上下文注入 token 预算，提升推理吞吐。
- 降低因隐式覆盖导致的一致性错误。
- 提升审计性，支持从摘要快速定位原始证据。

上述效果可通过压缩比、冲突簇率、平均检索耗时、注入 token 数等指标量化验证。
