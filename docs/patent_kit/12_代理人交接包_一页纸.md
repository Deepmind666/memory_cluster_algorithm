# 12 代理人交接包一页纸

## 1. 项目一句话
本案是一个面向多 Agent 记忆管理的“冲突感知聚类压缩”方案，核心不在通用聚类本身，而在“冲突可计算治理 + 合并风险阻断 + 预算自适应保真 + 语义作用域精确抽取”的组合机制。

## 2. 建议主打的核心创新点（建议写入主权项）
1. 冲突证据图 CEG：将同槽位多值冲突结构化为节点/边/证据计数并计算优先级。
2. 自适应保留预算 ARB：按冲突密度、来源分布、时效衰减联合分配摘要预算。
3. 双通道合并门控 DMG：语义相似通道与冲突兼容通道同时通过才允许合并。
4. 冲突语义精度抽取：条件/反事实/否定作用域与跨句回指联合处理，降低误抽取。

## 3. 当前权利要求策略
